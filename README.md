# Nixo FDE Slackbot

> A real-time Slackbot and dashboard for Forward-Deployed Engineers to track customer support tickets, bugs, and feature requests. 

**Built by:** Advay

### Backend Stuff

```bash
cd backend
npm install
cp .env.example .env
# Edit .env !!
npm run dev
```

The backend will run on `http://localhost:3000`
